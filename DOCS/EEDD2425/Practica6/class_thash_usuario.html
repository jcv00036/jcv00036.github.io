Calcula la posición en la tabla hash utilizando una función de dispersión cuadrática.

La función que aplicamos es la fórmula: clave + (i^2) % _maxElementos para calcular la posición en la tabla hash.

Parámetros
claveClave numérica que se dispersa
iIntento de colisión
Devuelve
unsigned long Posición final calculada en la tabla hash.

Calcula la posición en la tabla hash utilizando dispersión doble (versión 1).

La función usa la fórmula: posicion + intento * (_primoMenorAtributo - clave % _primoMenorAtributo) % _maxElementos
